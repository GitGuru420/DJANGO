***Create virtual environment and initialize a project***
Create django project: 
1. Create virtual environment.
    -> python -m venv projectName_env
2. Environment activate.
    -> projectName_env\Scripts\activate
3. django install.
    -> pip install Django
    -> python.exe -m pip install --upgrade pip
Optional: check django install done or not? 
    -> pip list
4. create django project.
    -> django-admin startproject projectName
    -> Less Redundancy: django-admin startproject projectName .
Optional: check django project create done or not? 
    -> python manage.py runserver
